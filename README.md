# StatABench
The official code for StatABench

## Mcp agent
Execute the following command to run the Mcp agent:
```bash
python ./mcp_scripts/mcp_server.py # start the mcp server
python ./mcp_scripts/mcp_pipeline.py  # start the mcp agent experiment

```
## Other agents
You can find the code for testing other agents in the `agent_scripts` folder.
```bash
python ./agent_scripts/agent_autogen.py
```
